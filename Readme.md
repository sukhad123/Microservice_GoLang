# Go Web Server with PostgreSQL & Docker

## Overview

This project is a **Go web server** built with the `chi` router, connected to a **PostgreSQL** database using **sqlc**. It exposes RESTful APIs for managing products and is fully **dockerized** for easy deployment.

---

## Features

- REST API endpoints for products:
  - `GET /products` — list all products
  - `POST /products` — create a new product
- PostgreSQL integration with SQL queries generated by **sqlc**
- Structured service and handler layers
- Error handling and JSON responses
- Dockerized for consistent environment

---

## Tech Stack

- **Language:** Go  
- **Database:** PostgreSQL  
- **SQL Generation:** sqlc  
- **Containerization:** Docker
